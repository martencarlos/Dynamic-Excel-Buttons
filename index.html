Excel-button

VBA code that creates listeners cells that, when changed, creates a button on adjacent right cell. When clicked open path to the specified word file

Automatic Excel button generation

I'm starting to work more seriously with Excel files and have encountered a big frustration when I wanted to create a button that, when pressed opened a word document. As it turns out it requires quite a bit of coding.

For this reason I decided to go through the process of creating a robust and standard way to create buttons in Excel, and posted on Github so that anyone who feels the same need and has the expertise can contribute to the project.

To get this feature, simply download the latest version of the Excel file from the Releases tab. Once you open the file there will be instructions and tips on how to create the buttons.

I really hope this helps you with your frustration of creating buttons in Excel :)
